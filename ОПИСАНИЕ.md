# CRM GoodPeople — описание проекта

## Что это за проект

`CRM` — это платформа для кадрового агентства, которая автоматизирует путь от заявки клиента до закрытия вакансии и работы с кандидатами.

Система состоит из:
- публичной части (лендинг + форма заявки);
- CRM-интерфейса для менеджеров и администраторов;
- backend API на FastAPI;
- фоновых задач;
- модулей календаря, матчей кандидатов, уведомлений и договоров;
- PostgreSQL + Redis.

Ключевой сценарий:
1. Клиент оставляет заявку.
2. Менеджер обрабатывает заявку и фиксирует коммуникацию в CRM.
3. Менеджер ведет заявку в CRM (канбан, заметки, статусы).
4. Система помогает подобрать кандидатов и отправить предложения.
5. По заявке ведутся уведомления, календарь и документы.

## Технологический стек

- Frontend: `Next.js 14`, `React`, `TypeScript`, `Tailwind CSS`
- Backend: `FastAPI`, `SQLAlchemy`, `Alembic`
- Очереди/фоновые задачи: `Celery`, `Redis`
- БД: `PostgreSQL 16`
- Инфра: `Docker`, `Docker Compose`, `Nginx`
- CI: `GitHub Actions` (миграции + тесты backend + docker build check)

## Как запустить проект локально

## 1) Подготовка

1. Установить `Docker` и `Docker Compose`.
2. В корне проекта создать файл `.env` (можно взять за основу `.env.example`).
3. Обязательно заменить все ключи/пароли на свои значения.

Минимально важные переменные:
- `POSTGRES_DB`, `POSTGRES_USER`, `POSTGRES_PASSWORD`
- `DATABASE_URL`
- `SECRET_KEY`
- `REDIS_URL`
- `NEXT_PUBLIC_API_URL` (для frontend)

## 2) Запуск через Docker Compose

Из корня проекта:

```bash
docker compose up -d --build
```

Проверка статуса:

```bash
docker compose ps
```

## 3) Применить миграции БД

```bash
docker compose exec backend alembic upgrade head
```

## 4) Проверка сервисов

- Frontend: `http://localhost` (через `nginx`) или `http://localhost:3000`
- API docs: `http://localhost/api/docs` или `http://localhost:8000/api/docs`
- Health endpoint: `GET /api/health`

## 5) Полезные команды

Логи:

```bash
docker compose logs backend --tail=100 -f
docker compose logs frontend --tail=100 -f
```

Перезапуск:

```bash
docker compose restart
```

Остановка:

```bash
docker compose down
```
